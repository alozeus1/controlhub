# Brand Assets

This folder contains brand assets for Web Forx ControlHub.

## Usage

Import assets in React components:

```jsx
import logo from '../assets/brand/logo.svg';
import logoIcon from '../assets/brand/logo-icon.svg';
```

## Asset Specifications

| Asset | Size | Format | Usage |
|-------|------|--------|-------|
| `logo.svg` | 200x40 | SVG | Full logo with text |
| `logo-icon.svg` | 40x40 | SVG | Icon only (sidebar, favicon) |
| `logo-dark.svg` | 200x40 | SVG | Logo for light backgrounds |
| `logo-light.svg` | 200x40 | SVG | Logo for dark backgrounds |

## Brand Colors

- Primary: `#00D4FF` (Cyan)
- Primary Dark: `#0099CC`
- Background: `#0A0F1C`
- Text: `#F9FAFB`

## Typography

- Primary Font: Inter
- Monospace: JetBrains Mono
